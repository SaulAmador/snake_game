# Snake Game with Obstacles 🐍

[Leer en español](./README.es.md)

# Presentation

This project is my final submission for the **Code in Place** certification course taught by Stanford University.
The goal was to apply the concepts learned in programming and game design, adding creativity and improvements to the classic Snake game.

# Description

This is an improved classic Snake game, developed in Python using Turtle and Freegames, with the following features:

- Random obstacles that increase throughout the game and never overlap.
- Obstacles and food with realistic collision detection.
- Moving obstacles.
- Food can vary after a certain score.
- Visual effect when eating food (flashing).
- "Red flash" visual effect when losing.
- Game Over message with restart option.
- The snake changes color depending on its size.

# Requirements

- Python 3.x
- [free games](https://pypi.org/project/freegames/)

Install free games with:

```sh
pip install free games
```

# How to play

1. Run the Python file:

```sh
pythonsnake.py
```

2. Control the snake with the arrow keys.
3. Eat food to grow and score points.
4. Avoid colliding with obstacles, the edge, or yourself.
5. When you lose, press the `R` key to restart the game.

# Controls

- Up Arrow: Move up
- Down Arrow: Move down
- Left Arrow: Move left
- Right Arrow: Move right
- R: Restart the game after Game Over
